欢迎使用 风的工具箱

        
这是一个“抓包转脚本”的生成器，不是直接跑空投机器人。

        
    
          
  1. 准备一个抓包文件（HAR 优先）。
    2. 
          
  2. 在新手向导里点“开始自动分析”。
    3. 
          
  3. 点“一键生成”，按状态区命令运行项目。
    4. 
        

        

          
        

      

    


    

      

        
不会抓包？按这个流程做（HAR 推荐）

        
    
          
  1. 打开目标网站，按 F12 打开开发者工具。
    2. 
          
  2. 切到 Network / 网络，勾选 Preserve log / 保留日志。
    3. 
          
  3. 点清空（垃圾桶图标），从“登录”开始完整操作一遍任务。
    4. 
          
  4. 请求列表空白处右键，点 Save all as HAR with content。
    5. 
          
  5. 保存得到 .har 文件，然后回软件里选择这个文件。
    6. 
        

        

          提示：HAR 可能包含 token/cookie，请勿发给他人，也不要上传公开仓库。
        

        

          
        

      

    


    

      

        
术语翻译（人话版）

        
    
          
  • 抓包文件(HAR)：你在网站上操作时，浏览器发出的接口记录。
    • 
          
  • Token：网站给你的“登录凭证”，有它就能继续请求接口。
    • 
          
  • 私钥：钱包控制权，必须保密，泄露就可能丢资产。
    • 
          
  • RPC_URL：连接区块链节点的网址，链上任务必须有。
    • 
          
  • Nonce：一次性随机串，签名登录前常会先拿它。
    • 
          
  • Claim：领取奖励。
    • 
          
  • Faucet：测试网领币。
    • 
          
  • 429：请求太频繁，被网站限流了。
