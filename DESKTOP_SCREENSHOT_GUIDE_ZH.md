# 桌面版截图流程（傻瓜式）

目标：从 GitHub 下载 `exe` 安装包，打开图形界面，生成第一个项目。

## 1. 下载安装包

1. 打开：`https://github.com/ranjing666/script-generator/releases`
2. 进入最新版本
3. 在 `Assets` 点击 `ScriptGenerator-<version>-Setup-x64.exe`

截图位 A：能看到 `ScriptGenerator-...-Setup-x64.exe` 文件名。

## 2. 安装软件

1. 双击 `ScriptGenerator-...-Setup-x64.exe`
2. 连续点击“下一步”
3. 点击“安装”
4. 点击“完成”

截图位 B：看到安装完成页面，带 `Finish`/“完成”按钮。

## 3. 打开软件

1. 双击桌面图标 `Script Generator`
2. 进入主界面，顶部可看到两个页签：
- `手动拼接`
- `抓包导入`

截图位 C：看到软件标题 `Script Generator Desktop`。

## 4. 用导入模式快速生成

1. 切换到 `抓包导入`
2. 点击 `导入文件路径` 右侧“选择”
3. 选择一个 `HAR` 文件
4. 点击 `分析抓包`
5. 保持默认勾选任务组
6. 点击 `生成项目`

截图位 D：分析结果区域出现“候选请求/任务组数量”等信息。

## 5. 打开输出目录

1. 点击 `打开输出目录`
2. 进入生成目录，确认存在：
- `project.config.json`
- `main.js`
- `data/`
- `README.md`

截图位 E：能看到以上文件名。

## 6. 常见卡点

- 点“分析抓包”报错：通常是文件路径不对，重新选择文件。
- 点“生成项目”报错：先确认至少勾选了一个任务组。
- 无法运行生成脚本：先按生成目录里的 `README.md` 安装依赖。

## 7. 安全提醒

- 不要把私钥/token/账号文件上传到公开仓库。
- 先用测试号、小并发跑通，再放大。
