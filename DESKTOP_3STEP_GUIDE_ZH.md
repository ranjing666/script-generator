# 桌面版 3 步上手（超简版）

目标：完全不懂脚本，也能在 3 步内生成可运行项目。

## 第 1 步：打开软件并保持“小白模式”

1. 启动 `风的工具箱v1.1.0`
2. 右上角确认 `小白模式` 已勾选
3. 切到 `抓包导入`

## 第 2 步：选抓包文件，点一键生成

1. 点击“选择”，选你的 `HAR / Postman / cURL` 文件
2. 点击 `自动识别 + 一键生成`
3. 等待提示 `一键生成成功`

软件会自动完成：
- 抓包类型识别
- 请求分析
- 任务组选择
- 项目生成

## 第 3 步：照着提示运行

生成成功后，状态区会给出下一步命令：

1. 进入输出目录
2. `npm install`
3. 复制 `.env.example` 为 `.env`
4. `npm start`

## 新手注意

- 第一次请先用小号测试
- 私钥/token 文件不要上传公开仓库
- 失败先看 `import.report.json` 的 `warnings`
